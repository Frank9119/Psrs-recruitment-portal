# Psrs-recruitment-portal
Full-stack recruitment system


## Project Structure
1. `psrs_accounts`:  
    User managements, Separation and profile managements. Hapa utaasign users kulingana na nafasi zao. Mfano: Admin, Management na category Admin.

1. `psrs_auth`: 
    User managements, Separation and profile managements. Hapa utaasign users kulingana na roles zao. Mfano: Admin, Applicants na Recuiter. 

1. `psrs_job`:   
    Manage all Jobs Applications. 

1. `psrs_settings`:  
    Settings zote za mfumo zinafanyika hapa. kitu chochote ambacho ni confugurable kinafanika hapa. Mfano: Application types.


## Running the Application

1. **Local host** `python manage.py runserver 0.0.0.0:8000`

### Database configurations
Mysql database is used.
